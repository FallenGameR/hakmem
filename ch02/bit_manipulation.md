# Bit manipulation

## turn_off_rightmost_1bit

`x & (x - 1)`

```text
 10101000
 10100000
 10000000
 00000000 - stays the same
 00100000 -> pow2
 00000000 -> all zeros
```

## turn_off_rightaligned_1bit_group

`x & (x + 1)`

```text
 10100111
 10100000 - stays the same
 00111111 -> pow2 - 1
 00000000 -> all zeros
 11111111 -> all ones
```

## turn_off_rightmost_1bit_group

`x & (x + (x & -x))`

```text
 01011100
 01000000
 00000000 - stays the same
```

## turn_on_rightmost_0bit


## turn_on_rightaligned_0bit_group

`x | (x - 1)`

```text

```
